# 💚 Bump Brat

A TUI tool for bumping frontend production hashes in app-interface.

## What It Does

Bump Brat helps you update frontend project commit references to their latest versions. It:

- Shows all configured frontend projects with their current status
- Displays what commits will be included in each bump
- Automatically stages changes and generates commit messages
- Creates ready-to-use MR descriptions

## Quick Start

```bash
cd bump-tool
./bump-brat
```

Use arrow keys to navigate, press **enter** to bump a project, press **q** to quit.

## GitHub Authentication (Recommended)

Without authentication, you're limited to 60 GitHub API requests per hour. With a token, you get 5,000 requests per hour.

### One-Time Setup

1. **Create a token** at https://github.com/settings/tokens
   - Click "Generate new token (classic)"
   - Give it a name (e.g., "bump-brat")
   - Select scope: `public_repo`
   - Generate and copy the token

2. **Add to your shell config** (`~/.zshrc` or `~/.bashrc`):
   ```bash
   export GITHUB_TOKEN="ghp_your_token_here"
   ```

3. **Reload your shell**:
   ```bash
   source ~/.zshrc
   ```

### One-Off Usage

```bash
GITHUB_TOKEN="ghp_your_token_here" ./bump-brat
```

## Controls

| Key | Action |
|-----|--------|
| **↑/↓** | Navigate projects |
| **d** | View commit details |
| **enter** | Bump selected project |
| **q** | Quit |

## After Bumping

The tool provides two things to copy:

1. **Commit message** for your git commit
2. **MR body** with all the changes included

Example output:
```
📋 COMMIT MESSAGE
--------------------------------------------------------------------------------
landing-page-frontend: bump to def5678
--------------------------------------------------------------------------------

📋 MERGE REQUEST BODY
--------------------------------------------------------------------------------
Bumps landing-page-frontend from abc1234 to def5678

- Fix layout issue on mobile (sha1)
- Update dependencies (sha2)
- Add new feature (sha3)

---
🤖 Generated by bump-brat
--------------------------------------------------------------------------------
```

## Adding New Projects

Edit `bump-brat-config.yml`:

```yaml
projects:
  - name: your-frontend
    file_path: ../data/services/insights/your-service/deploy.yml
    repo_url: https://github.com/RedHatInsights/your-frontend
    line1: 123  # First ref: line number
    line2: 129  # Second ref: line number
```
